# Online Retail – RFM & Customer Retention (Power BI)

End-to-end Power BI dashboard built on the public **Online Retail (UCI)** dataset (Dec 2010 – Dec 2011).

## Objective
- Analyse **customer behaviour and retention**.
- Build an **RFM model** (Recency, Frequency, Monetary) and customer segments (Champions, Loyal, High-Risk, Lost, Other).
- Estimate **CLV** and identify **revenue at risk**.
- Track **monthly performance** (revenue, active customers, repurchase rate, MoM growth).
- Understand **product performance** by category, country and top products.

## Main report pages
1. **Executive Summary**
2. **Customer Segmentation & RFM**
3. **RFM Retention & Key Insights**
4. **Product Performance**
5. **Monthly Performance & Trends**
6. **Documentation** (data model, business rules, assumptions)

## Files
- `Online-Retail-RFM-PowerBI.pbix` – main Power BI report.
- `screenshots/` – selected report pages.

## Tools
- Power BI Desktop  
- Power Query  
- DAX  
- Excel  

## How to use
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop**.
3. Use the navigation buttons at the top of each page to move between report sections and apply slicers (date, country, segment, etc.).
